# 我的第一篇博客：学习与开源记录

大家好！我是**硫氢化钠**，这是我的个人主页和博客。我打算在这里记录我的**学习经验**，以及参与**开源项目**的心得。

## 为什么搭建这个博客？

作为一个普通的大学生，我也许与大多数人一样，大一大二不知道怎么就过去了。但是真正有用的知识却没学到多少。想要学习时，也因迟迟不知道该如何学习，从何处下手而导致拖延。可能是某一天我幡然醒悟，在这个AI大时代，我们的学习可能并不像高中的应试教育一样，更多的是利用现在的工具来实现我们天马行空的想法。当一个小成功展示在你的眼前时，一个种子将会在你的心中悄然发芽，它有着大大的能量，有着一种让人一往无前的动力。当然，学会使用工具固然重要，但是自身的基础也非常关键，这是决定你上限和能否利用好工具的关键因素。我发现，当我将我自己所学会的内容教给别人的时候，哪怕只有那么一两个人可以听懂并驻足观看，我认为都是值得的。同时这也是一个将我自身知识融汇贯通的良好方法。所以，有一个记录自己成长的平台就显得非常重要了，这个博客的作用有：

- **积累知识**：记录自己学习过程中遇到的问题，以及解决方法。
- **开源分享**：把自己的*开源项目*展示出来，让更多人看到并参与。
- **经验总结**：记录学习过程中的踩坑与心得，帮助自己，也希望能帮助到别人。

## 博客技术栈

目前这个博客非常极简，采用**纯静态 HTML/CSS + 原生 JavaScript** 构建：
- 零构建工具，随改随看
- 借助 `marked.js` 实现 Markdown 实时解析
- 使用 `highlight.js` 实现代码高亮
- 原生 CSS 变量实现暗黑模式
